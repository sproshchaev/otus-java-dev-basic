# otus-java-dev-basic
Examples from webinars

[ListDemo01.java](src/main/java/com/prosoft/ListDemo01.java) - примеры работы с интерфейсом List и реализаций ArrayList

[ListDemo02.java](src/main/java/com/prosoft/ListDemo02.java) - создание и заполнение списков строк  

[ListDemo03.java](src/main/java/com/prosoft/ListDemo03.java) - основные методы работы с ArrayList: добавление, получение, замена и удаление элементов, проверка содержимого, управление емкостью списка и получение его размера   

[ListDemo04.java](src/main/java/com/prosoft/ListDemo04.java) - основные операции с LinkedList: добавление элементов в начало и конец, получение, замену и удаление элементов, проверку содержимого, а также очистку списка   

[ListDemo05.java](src/main/java/com/prosoft/ListDemo05.java) - демонстрирует особенности работы с ArrayList, включая обработку исключений при обращении к несуществующим индексам, поведение при удалении и добавлении элементов, а также ограничения на операции с индексами за пределами текущего размера списка   

[ListDemo06.java](src/main/java/com/prosoft/ListDemo06.java) - демонстрирует временную сложность метода get() у ArrayList, измеряя среднее время доступа к элементам по индексу при разных размерах списка, подтверждая постоянную сложность O(1)   

[ListDemo07.java](src/main/java/com/prosoft/ListDemo07.java) - демонстрирует временную сложность метода contains() у ArrayList, измеряя среднее время поиска элемента в списке различного размера и подтверждая линейную сложность O(n)   

[ListDemo08.java](src/main/java/com/prosoft/ListDemo08.java) - демонстрирует основные операции по созданию и работе с элементами списков ArrayList и LinkedList, включая добавление и получение элементов   

[ListDemo09.java](src/main/java/com/prosoft/ListDemo09.java) - демонстрирует способы удаления элементов из ArrayList: использование итератора для безопасного удаления во время обхода коллекции, а также метод removeIf для удаления элементов на основе заданного условия
